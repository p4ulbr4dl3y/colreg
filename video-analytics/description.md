# Описание проекта: «Автономный навигационный комплекс БЭС с поддержкой МППСС-72»

**Цель проекта:** Создание программно-аппаратного комплекса для автоматического обнаружения, классификации морских объектов и генерации рекомендаций по расхождению для удаленного оператора в строгом соответствии с правилами МППСС-72.

### Архитектура системы (Hardware & Software)
1. **Навигационный комплекс (Собственное судно):** GPS/ГЛОНАСС + Компас (IMU) — *обязательно для расчета истинного курса и скорости своего судна.*
2. **Радар (РЛС):** Источник первичного обнаружения (дальнобойный, работает в туман и ночью).
3. **PTZ-камера:** Оптическая система с изменяемым углом поворота, наклона и зумом (управляется по протоколу ONVIF/VISCA).
4. **Модуль видеоаналитики (Уже готов):** Нейросеть (вероятно, YOLO + классификатор), принимающая RTSP-поток с камеры.
5. **Центральный хаб (Шина данных):** Брокер сообщений (рекомендуется ROS 2 или MQTT) для обмена данными между модулями в реальном времени.
6. **Модуль принятия решений (МПР):** Ядро логики. Анализирует векторы движения и типы судов, выдает маневр.
7. **АРМ Оператора (Пульт):** Интерфейс, где отображается радарная обстановка, видео с рамками аналитики и текстовая/графическая подсказка от МПР (например: *"Опасность столкновения! Правило 15. Уклонитесь вправо на 20 градусов"*).
